# 🤖 FRIDAY AI Assistant

FRIDAY is an AI-powered chatbot built using Python, Streamlit, and Google Gemini API.

## Features

* AI-powered conversations using Gemini
* Modern Streamlit interface
* Chat history support
* Secure API key management using `.env`
* Custom FRIDAY assistant personality

## Technologies Used

* Python
* Streamlit
* Google Gemini API
* Git & GitHub

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
